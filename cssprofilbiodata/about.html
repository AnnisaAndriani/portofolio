Cerita Singkat Saya

Saya menyelesaikan pendidikan SMK dengan jurusan rekayasa perangkat lunak pada tahun 2021. Setelah itu, saya memilih untuk melanjutkan pendidikan di Universitas Terbuka karena ingin berkuliah sambil bekerja. Di Universitas Terbuka saya mengambil jurusan ilmu perpustakaan karena melihat peluang pekerjaan yang besar setelah lulus.

Selama masa kuliah, saya juga melanjutkan usaha keluarga di pasar. Pekerjaan ini tidak membuat tugas kuliah saya terbengkalai. Selain itu, untuk lebih mendalami bidang yang saya pilih, saya mencoba membantu pekerjaan saudara saya yang berprofesi sebagai pustakawan di perpustakaan Universitas Musi Rawas.

Di saat libur kuliah, saya mengisi waktu dengan mengikuti beberapa kelas menulis online. Pengalaman kuliah sambil bekerja ini membuat saya lebih terampil dalam manajemen waktu, berpikir kritis, dan menghasilkan karya.
